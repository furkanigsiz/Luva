# Luva: AI-Powered Gmail Classifier & Intelligent Email Assistant

Luva (Priority Inbox) is an advanced, open-source **AI Email Classifier** and **Gmail Productivity Tool** that transforms your cluttered inbox into a sleek, organized dashboard. Powered by a hybrid **Go/Python** architecture and state-of-the-art **LLMs**, Luva automatically categorizes, prioritizes, and summarizes your emails in real-time.

## Advanced Features of the Luva AI Assistant

*   **Multi-Provider AI Intelligence**: Seamlessly switch between **Google Gemini**, **OpenAI**, and **OpenRouter** models to power your email classification.
*   **Dynamic Language Support**: Full internationalization support with **Turkish (TR)** and **English (EN)** localization modes.
*   **Intelligent Prioritization**: Uses **Natural Language Processing (NLP)** to rank emails from Critical (Priority 1) to Minimal (Priority 5), helping you focus on what truly matters.
*   **Smart Categorization**: Automatically organizes threads into Work, Billing, Social, Personal, and Promotion columns.
*   **Interactive Setup Wizard**: An integrated, step-by-step guide for configuring Google Cloud projects, Gmail API, and OAuth credentials directly within the app.
*   **Premium Earthy Aesthetic**: A high-end, eye-friendly "Obsidian Earth" theme featuring glassmorphism, animated mesh gradients, and a responsive grid layout.
*   **Account & Data Management**: Full control over your Gmail connection, cache management, and personal configuration through a dedicated account portal.

## Why Choose Luva for Gmail Automation?

 Luva goes beyond simple filtering by providing an **AI-driven summary** for every message. It acts as your personal **Open Source AI Agent**, reducing the time spent on manual sorting and helping you achieve "Inbox Zero" with minimal effort.

## Getting Started

### 1. Installation

Install the **AI content analysis** engine dependencies:
```bash
pip install -r requirements.txt
```

Start the **Go-based web dashboard**:
```bash
go run main.go
```

### 2. Configuration (The Luva Setup Wizard)
Luva simplifies the complex Google API setup through its built-in **Setup Wizard**:
- **Google Cloud Integration**: Step-by-step guidance for enabling Gmail API.
- **OAuth Setup**: Direct input for Client ID and Client Secret from within the dashboard.
- **AI Configuration**: Support for multiple API keys with secure, toggleable visibility.

## Technical Stack for Open Source Contributors

*   **Backend engine**: Python for LLM integration and deep message analysis.
*   **System Core**: Go (Golang) for the high-speed proxy server and internal routing.
*   **Frontend**: Vanilla JS, CSS3 (Glassmorphism), and HTML5 with a focus on premium aesthetics and SEO.
*   **Security**: OAuth 2.0 protocol for localized, secure Gmail access.

## Use Cases

*   **Executive Productivity**: Grasp the gist of 50+ emails in minutes via AI summaries.
*   **Customer Support**: Isolate "Critical" client requests automatically.
*   **Tech-Focused Inbox**: Filter technical alerts and repository notifications into specific columns.
